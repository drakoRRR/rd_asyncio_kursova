# CVE REST API, Scrapper

## Installation 
1. Fill .env file by .env .example:
```angular2html
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
DB_HOST=
DB_PORT=

POSTGRES_DB_TEST=
DB_TEST_HOST=
POSTGRES_TEST_USER=
POSTGRES_TEST_PASSWORD=
DB_TEST_PORT=
```
2. use `docker-compose up -d --build`

By this url you can achieve API Documentation: http://127.0.0.1:5000/api/docs
